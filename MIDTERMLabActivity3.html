What is HTML?

HTML stands for HyperText Markup Language. It is used to structure content on the web,
telling browsers how to display text, images, links, and other media. HTML isnâ€™t a
programming language, but a markup language, which means it is used to annotate and format content.



How HTML Works

HTML uses a system of tagsand attributes to structure and present content on the web. Tags are enclosed in angle brackets, Most HTML elements have an opening tag and a closing tag, though some (like images and line breaks) are self-closing.

Web browsers read these tags and use them to render content visually. For example, the <h1> tag displays large, bold headings, while the <a> tag creates a hyperlink. Attributes provide extra information about an element, like the source of an image or the destination of a link.



Common HTML Tags

  • <html>: Defines the HTML document.
  • <head>: Contains metadata such as the title and character set.
  • <title>: Sets the title of the web page (appears in the browser tab).
  • <body>: Contains the content of the webpage.
  • <h1> to <h6>: Headings, with <h1> being the highest.
  • <p>: Defines a paragraph of text.
  • <a>: Creates hyperlinks to other pages.


HTML Attributes

HTML elements can have attributes that provide additional information. For example:

  • href: tag to define the URL of the link.
  • src: tag to define the source of an image.
  • alt: Provides alternative text for images in case the image cannot be displayed.
  • class: Allows you to apply CSS styles to elements with a specific class.
  • id: Uniquely identifies an element within the document.
